School Projects

Here are a few examples of projects that were a part of my school curriculum that I am proud of.

Designed a low-pass filter

In Circuits 2, I designed a filter to accept or reject input signals. The filter was designed to reject input frequencies higher than the established set breakpoint frequency and accept those that were lower. This included ordering the parts needed (resistors, capacitors, op amp), using P-spice to make sure the math checks out, and using Eagle PCB design software to layout the low pass filter on a PCB board. I sent the board layout to be printed off on a PCB board. I finished out the process by soldering all the hardware components in their correct position. You can see the full lab report by clicking on the image to the left.

Programmed a BASYS 3 board to display my name scrolling

In Intro into Digital and Logical Design, I programmed a BASYS 3 board using the VHDL language. We used Vivado to program the board. The end result was my name scrolling from left to right when one of the switches were flipped. You can read about the project by clicking on the image to the left.

Maze walking program in python

In Programming 2, we were tasked with a maze walking problem when given a text file input that would show a "maze". The maze was marked with "P" (people), "S"(street), "@"(sewers), and "#"(buildings). The goal was to see how a "blob" (marked "B") would spread through a city, if the blob could only move through streets ("S") and people ("P"). The sewers ("@") allowed the blob ("B") to transport to a different part of the maze. The buildings ("#") were walls that the blob could not pass. You can see this project on GitHub by clicking on the image to the left.
